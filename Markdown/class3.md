# 📝 Streamlit 點餐機筆記（簡單版）

## 1️⃣ 開啟程式

- 我們要做一個「點餐機」，可以輸入餐點、加入購物籃、還可以刪除。
- `st.session_state` 就像一個小箱子，可以存東西（這裡存的是購物籃裡的餐點）。

```python
if "cart" not in st.session_state:
    st.session_state.cart = []  # 建立一個空的購物籃
```

---

## 2️⃣ 重新整理按鈕

- 按下「重新整理」，畫面會更新，像刷新網頁一樣。

```python
if st.button("重新整理"):
    st.rerun()  # 重新跑一次程式
```

---

## 3️⃣ 輸入餐點 + 加入購物籃

- 輸入餐點名稱，按「加入」，餐點就會跑到購物籃裡。
- 用 `st.columns` 可以把畫面分成兩邊：左邊輸入，右邊按按鈕。

```python
food = st.text_input("輸入餐點")
if st.button("加入"):
    if food != "":
        st.session_state.cart.append(food)  # 把餐點放進購物籃
        st.rerun()  # 更新畫面
```

---

## 4️⃣ 顯示購物籃

- 購物籃裡有東西才顯示。
- 每個餐點旁邊有「刪除」按鈕，按下去就會被移除。

```python
for i, item in enumerate(st.session_state.cart):
    st.write(item)
    if st.button("刪除", key=f"del_{i}"):
        st.session_state.cart.pop(i)  # 移除餐點
        st.rerun()  # 更新畫面
```

# 🧮 算數指定運算子筆記（小學生版）

這部分是**數字的快速計算方式**：

| 運算  | 例子      | 解釋                              |
| ----- | --------- | --------------------------------- |
| `+=`  | `a += 1`  | a = a + 1 → 把 a 加 1             |
| `-=`  | `a -= 1`  | a = a - 1 → 把 a 減 1             |
| `*=`  | `a *= 2`  | a = a \* 2 → 把 a 乘 2            |
| `/=`  | `a /= 2`  | a = a / 2 → 把 a 除 2（會變小數） |
| `//=` | `a //= 2` | a = a // 2 → 把 a 除 2，只取整數  |
| `%=`  | `a %= 2`  | a = a % 2 → 取 a 除 2 的餘數      |
| `**=` | `a **= 2` | a = a \*\* 2 → 把 a 平方          |

---

### 🌟 範例計算

```python
a = 1
a += 1  # 1+1 = 2
a -= 1  # 2-1 = 1
a *= 2  # 1*2 = 2
a /= 2  # 2/2 = 1.0
a //= 2 # 1//2 = 0
a %= 2  # 0%2 = 0
a **= 2 # 0**2 = 0
```

💡 小技巧：

- `+=, -=, *=` 就像**在原來的數字上再加減乘除**，很快又方便。
- `%` 是**餘數**，比如 5 % 2 = 1（5除以2剩1）。
